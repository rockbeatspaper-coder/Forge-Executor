# Forge-Executor
THIS PROJECT IS CLOSED SOURCE! THE GITHUB REPO IS ONLY FOR HANDLING AUTO UPDATES.




Forge Executor is a modern Roblox script execution client with a clean interface and a focus on stability.
The project is closed source.
This repository only exists to handle automatic updates for the client.

Features
Internal script execution

External tools panel

Script editor with syntax highlighting

Script hub integration

Dashboard and settings pages

Clean and responsive UI

Notes
The source code is not public

The client receives updates through this repository

Do not open issues requesting the source code

Installation
Download the latest release from the Releases section.
Run the installer and follow the on‑screen instructions.
